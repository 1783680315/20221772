# 📝 第04周实验报告：机器狗仿真程序运行
## 1.实验目标
在Ubuntu环境下成功启动机器狗（Quadruped Robot）物理仿真程序。

验证机器狗的URDF模型在仿真环境中的物理特性。

测试基础脚本，观察机器的初始姿势。

## 2.核心任务说明
环境加载：调用仿真引擎（如PyBullet）并加载机器狗的机械结构模型。

参数调试：调整仿真步长与重力参数，确保物理模拟的真实性。

交互控制：通过命令行或脚本驱动机器狗的关节，观察其反馈运动。
