# 📝 第05周实验报告：机械臂运动学仿真
## 1.实验目标
在PyBullet环境中加载多自由度机械臂模型。

理解并实践机械臂的正向运动学（正向运动学）与逆向运动学（逆向运动学）。

通过GUI界面手动调节关节参数，观察机械臂终止执行器的位置姿态变化。

## 2.核心任务说明
模型加载：在仿真窗口中成功渲染机械臂与桌面环境，并配置GUI调试参数。

关节控制（Joint Control）：通过调节右侧旋转（如10_Base，11_Shoulder等），直接控制各电机转动。

逆运动学调试（IK）：设置目标坐标target_X/Y/Z，观察算法如何自动计算各关节角度到到达指定位置。

视觉数据观察：观察窗口窗口中的RGB、深度图(Depth)和语义分割(Segmentation)数据流。

## 4. 关键实验参数

## 5.实验总结
本周的实验让我观察理解了机械臂在三维空间中的运动逻辑。通过手动调节IK目标，我发现当目标点超出机械臂的工作空间（Workspace）时，模型会出现眨眼或无法到达的情况。这为后续学习机械臂的路径规划打基础。
