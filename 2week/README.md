📝 第02周实验报告：OpenClaw本地部署
1.实验目标
在本地环境中完成OpenClaw开发开源项目的部署。

配置相关依赖库，确保代码能够成功编译并运行。

模型加载：在仿真器中成功加载OpenClaw的URDF模型。

2주차 실습 보고서: OpenClaw 로컬 배포

1. 실습 목표
OpenClaw 오픈소스 프로젝트를 로컬 환경에 배포합니다.

코드가 성공적으로 컴파일 및 실행될 수 있도록 관련 종속 라이브러리를 구성합니다.

2. 핵심 작업 설명

코드 복제: 공식 저장소에서 OpenClaw 소스 코드를 복제합니다.

환경 구성: Python 종속 라이브러리 또는 C++ 컴파일 환경을 설치해야 합니다.

모델 로딩: OpenClaw URDF 모델을 시뮬레이터에 성공적으로 로드합니다.
